Laptop Connection Guide
Physical Connection

Connect your laptop via Ethernet cable to the controller's secondary port or the local unmanaged switch in the panel. Do not unplug the primary network uplink without verifying site redundancy.

Unknown IP? How to find it (Discovery)

If the IP is not labeled on the panel door or controller:

  • Physical Check: Look for a sticker on the JACE/Controller casing.
  • Wireshark (Best): Install Wireshark. Connect to the port. Filter by arp. Watch for broadcast traffic ("Who has?"). The source IP is likely the controller or gateway.
  • Command Prompt: Run arp -a to see devices your laptop has recently talked to (only works if on same subnet).
Laptop IP Configuration

Most BAS networks use Static IPs. Do not use DHCP unless confirmed.

Method A: The Shortcut (Fastest)
  1. Press Win + R, type ncpa.cpl, hit Enter.
Method B: Windows Search
  1. Press Start (Windows Key).
  2. Type "View network connections" and hit Enter.
Next Steps: Right-click Ethernet → Properties → Select IPv4 → Properties → "Use the following IP address".
Example Configuration:
Target Controller IP: 192.168.1.140
Set Laptop IP: 192.168.1.200 (Last octet must be unique)
Subnet Mask: 255.255.255.0
Verify & Find Ports

Open PowerShell to verify connectivity and scan for open ports if the web interface is not loading.

# 1. Ping the controller
ping 192.168.1.140

# 2. Check for BACnet (47808)
Test-NetConnection 192.168.1.140 -Port 47808

# 3. Check for Niagara/Fox (1911)
Test-NetConnection 192.168.1.140 -Port 1911
